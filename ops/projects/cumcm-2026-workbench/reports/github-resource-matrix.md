# GitHub Resource Matrix

This report records repositories and skill sources worth localizing before the
contest starts. It is a pre-contest preparation artifact, not a contest-day
browsing checklist.

## Template Sources

- `latexstudio/CUMCMThesis`
  - Use for local LaTeX template snapshots aligned with CUMCM-style structure.
  - Localize with `python3 src/main.py fetch-template` before contest day.
  - Keep it under ignored `templates/vendor/`; the current checked snapshot has no explicit top-level license file, so do not vendor it into the workbench repo by default.
- `chenboshuo/cumcm_template`
  - Use for workflow ideas around split files, scripts, and template assembly.

## Skills / Workflow References

- `openai/skills`
  - Official Codex skill source for installation and local adaptation ideas.
- `ndpvt-web/latex-document-skill`
  - Useful reference for LaTeX-oriented workflow automation.
- `RichradsY/PaperSprint`
  - Useful reference for paper polishing and revision loop ideas.

## Offline-First Guidance

- Localize useful templates and notes before `2026-09-10`.
- Do not use GitHub to browse contest-problem discussions after the contest
  starts.
